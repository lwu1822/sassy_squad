Nesting

While writing CSS, you may notice that many selectors share the same element.

For example, let's say you have a div element that controls the font. Inside the div element, you have two other divs that control the font size.

Your CSS code might look this: 

            
.font .title { 

            
	font-family: "Times New Roman", serif; 

            
	font-size: 3em;

            
}

            
.font .text {

            
	font-family: "Times New Roman", serif; 

            
	font-size: 1em;

            
}

SASS provides a feature called nesting in which you can write styling code in a way that looks like an HTML hierarchy.

The SASS equivalent for the CSS code above would be:

            
.font {

            
	.title {

            
	font-family: "Times New Roman", serif;

            
	font-size: 3em;

            
	.text {

            
	font-family: "Times New Roman", serif;

            
	font-size: 1em;/div>
            
}

            
}

Below is an example of HTML using the SASS code and its output:

            
<div class="font">

            
	<div class="title">

            
	<p>Title</p>

            
	</div>

            
	<div class="text">

            
	<p>This is some text</p>

            
	</div>

            
</div>

(Output using the SASS and HTML code above)

Title

This is some text
